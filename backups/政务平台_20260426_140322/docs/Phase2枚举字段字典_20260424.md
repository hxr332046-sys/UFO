# Phase 2 枚举字段字典（business → code）

> **日期**：2026-04-24
> **目的**：把 `docs/资料.txt` 里用人话写的"自有产权/租赁"、"男/女"、"党员/群众"等选项，映射到服务端 HTTP body 里的真实字符编码（`"01"/"02"`、`"1"/"2"`、`"01"/"13"`）
> **代码入口**：`@/UFO/政务平台/system/phase2_enums.py`（含 helper 函数）
> **数据来源**：`dashboard/data/records/establish_save_samples/*.json`（mitm 实录）+ `_archive/_phase2_mi_save_v7.py`

---

## 一、使用方式

### 1.1 写 case 时（人话）

`docs/case_有为风.json`：
```json
{
  "property_use_mode": "租赁",
  "house_to_bus": "否",
  "politics_visage": "群众",
  "person": { "name": "黄永裕", ... }
}
```

### 1.2 body 构造时（Python helper）

```python
from phase2_enums import house_to_bus, use_mode, politics_visage

body["houseToBus"] = house_to_bus("否")        # → "0"
body["useMode"] = use_mode("租赁")              # → "02"
member["politicsVisage"] = politics_visage("群众")  # → "13"
```

### 1.3 兜底策略

- 人话**全字匹配** → 返回对应 code
- 人话**子串匹配** → 返回对应 code
- **都不匹配** → 返回默认值 + 打印 WARN（永不抛异常）

## 八、扩展清单（未来新增枚举时看这里）

添加新枚举字段的 checklist：

1. **抓实录**：在 mitm 里找到该字段的真实 code（不要猜）
2. **加字典**：`@/UFO/政务平台/system/phase2_enums.py` 新增 `FIELD_XXX` dict
3. **加 helper**：`@/UFO/政务平台/system/phase2_enums.py` 加一个 `field_xxx(value)` 函数
4. **加自测**：函数末尾的 `__main__` 块加一行测试
5. **写 MD**：本文件（`docs/Phase2枚举字段字典_20260424.md`）加一节
6. **连上 body 构造器**：`@/UFO/政务平台/system/phase2_bodies.py` 相应 build_xxx 里调 helper
7. **case 字段**：如果要允许用户配，加到 `case_xxx.json` 示例 + `Phase2Case` schema
